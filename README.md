## 📖 About

**NEO DOCS** is a powerful documentation explorer that allows you to browse and view project files with style. Built with FastAPI backend and a modern glass-morphism UI, it provides seamless navigation through your filesystem with beautiful syntax highlighting and markdown rendering.

### ✨ Key Features

- 🚀 **Full Filesystem Navigation** - Browse any directory on your system
- 📝 **Markdown Support** - Beautiful markdown rendering with syntax highlighting
- 🎨 **Modern UI** - Glass-morphism design with neon cyan-green accents
- 🔍 **Smart Search** - Search files with instant filtering
- 📁 **Directory Selector** - Modal-based filesystem explorer
- 💾 **File Operations** - Copy content, download files, copy paths
- 📌 **Recent Files** - Quick access to recently viewed files
- 🖼️ **Syntax Highlighting** - Support for 15+ programming languages
- 🌐 **Cross-platform** - Works on Windows, Linux, and macOS
